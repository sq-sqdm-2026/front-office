🎮 Gameplay Settings
Scouting Mode
Controls how accurate player ratings are when you view them.
Traditional — Your scouts estimate ratings with some uncertainty. Better scouting budget = more accuracy. Ratings are stable but may not be exact.
Stat-Based — Ratings are derived directly from a player's stats. No guesswork, but they only reflect past performance, not potential.
Variable — Ratings fluctuate daily based on recent performance. A hot streak boosts ratings; a slump lowers them. Most realistic but hardest to evaluate.
Difficulty
Affects your team's revenue, trade success rates, player development, and AI competitiveness.
Fan — Easiest. Boosted revenue, favorable trades, and faster player development. Great for learning.
Coach — Moderate. Slight advantages to help you compete. Good for casual play.
Manager — Balanced. No bonuses or penalties. The default experience.
Mogul — Hardest. Reduced revenue, tougher trades, and slower development. For experienced players who want a real challenge.
🔧 Advanced
Commissioner Mode
God mode. When enabled, you can directly edit any player's ratings, contracts, ages, and team finances. Use the player card or team finance page to make edits. Changes cannot be undone (except by loading a save).
Expansion Draft
Add a brand new franchise to the league and stock its roster via an expansion draft. Only available during the offseason.
CSV Export / Import
Export your data to spreadsheets (CSV format), or import data to overwrite current values. Useful for custom edits or backups.
🤖 AI Models (Ollama)
Connection
The game connects to Ollama (a free, local AI app) for smarter GM personalities, scouting reports, and trade talks. The game works fine without it — it just uses simpler template text instead.
Ollama Address
Usually http://localhost:11434. Only change this if you installed Ollama on a different computer.

AI Models
If Ollama is connected, these models power different features. Install them by running the commands below in your terminal.
qwen3:32b — Smarter decisions (GM trades, negotiations)
Install: ollama pull qwen3:32b • ~18GB download • Needs 24GB+ RAM
qwen3:14b — Written content (scouting reports, news, messages)
Install: ollama pull qwen3:14b • ~8GB download • Needs 16GB+ RAM
Optional Models (Phase 3+)
These models enable advanced features like text-to-speech podcasts and image generation.
Orpheus TTS 3B — Audio podcast rendering (converts weekly podcast scripts to realistic multi-voice audio)
ollama pull orpheus-3b • ~1.7GB • Text-to-speech model for podcast audio
Used for: Weekly podcast audio, game broadcast commentary
Stable Diffusion (via ComfyUI) — AI-generated player portraits and baseball card art
Requires separate ComfyUI installation • pip install comfyui
Recommended model: sd3.5-medium (~4GB) or sdxl-turbo (~6GB)
Used for: Photorealistic player headshots, baseball card designs
Note: Without this, the game uses procedural pixel-art SVG portraits instead
Kokoro TTS — Alternative lightweight TTS for game broadcast play-by-play
pip install kokoro • ~300MB • Fast, runs on CPU
Used for: Real-time play-by-play audio during 8-bit game viewer
⚠️ Danger Zone
These actions are destructive and cannot be undone. Make sure to save your game first if you want to keep your progress.
Apply Schema Updates
Adds new database columns and tables needed by updated versions of the app. Safe — this does not delete your save data.
Reseed with Real MLB Rosters
Completely wipes your current database and rebuilds it from scratch using real MLB player data cached on disk.
⚠️ This will DELETE all your progress — your roster moves, trades, draft picks, financial decisions, and game history will be permanently lost.

Force Re-fetch from MLB API
Re-downloads all player data from the MLB Stats API (requires internet), then reseeds the entire database. Use this if player data seems outdated or corrupted. Takes 3-5 minutes.
⚠️ This will DELETE all your progress and re-download everything from scratch. Even slower than a regular reseed.
