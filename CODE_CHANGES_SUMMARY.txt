FILE 2: src/api/routes.py
================================================================================
Added 13 new API endpoints (~450 lines of code)

SECTION A: DEPTH CHART ENDPOINT
--------
Endpoint: GET /team/{team_id}/depth-chart
- Fetches all players for team
- Organizes by 11 positions
- Sorts by overall rating
- Returns JSON with position depth chart

SECTION B: COMMISSIONER MODE ENDPOINTS (7 total)
--------
1. POST /settings/commissioner-mode
   - Toggles commissioner mode on/off
   
2. GET /settings/commissioner-mode
   - Returns current state
   
3. POST /commissioner/edit-player/{player_id}
   - EditPlayerRequest model
   - Validates and clamps field values
   - Updates player ratings/personality
   
4. POST /commissioner/edit-team/{team_id}
   - EditTeamRequest model
   - Updates team finances and budgets
   
5. POST /commissioner/force-trade
   - ForceTradeRequest model
   - Executes trade without AI approval
   - Logs transaction
   
6. POST /commissioner/force-sign/{player_id}
   - ForceSignRequest model
   - Signs free agent with contract
   - Logs transaction
   
7. POST /commissioner/set-record/{team_id}
   - SetRecordRequest model
   - Placeholder for setting W-L record

SECTION C: STAT COLUMN ENDPOINTS (2 total)
--------
1. GET /settings/stat-columns
   - Returns current configuration
   - Provides default if not set
   
2. POST /settings/stat-columns
   - StatColumnConfig model
   - Saves batting/pitching column preferences

SECTION D: CSV EXPORT ENDPOINTS (4 total)
--------
1. GET /export/roster/{team_id}
   - Returns roster CSV (16 columns)
   - Streaming response
   
2. GET /export/batting-stats?season=2026
   - Returns batting stats CSV (18 columns)
   - Calculates AVG, OBP, SLG, OPS
   - Streaming response
   
3. GET /export/pitching-stats?season=2026
   - Returns pitching stats CSV (18 columns)
   - Calculates ERA, WHIP, K/9, BB/9
   - Converts IP to baseball format
   - Streaming response
   
4. GET /export/financials/{team_id}
   - Returns financial history CSV (16 columns)
   - Streaming response
